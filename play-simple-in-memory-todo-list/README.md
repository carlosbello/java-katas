# Simple in memory to-do list API using Play Framework

## Requirements

### To-do model

| Field | Type |
|-------|------|
| title | String |
| done | Boolean |
| creationDate | Date |

### Actions

* Add new
* Get all
* Get item
* Update
* Delete

## Execution

> sbt run
